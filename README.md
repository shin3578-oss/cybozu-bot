# サイボウズBot（cybozu-bot）

サイボウズの日報・週報をAIが添削し、自動承認するボットです。

## いつ動くか

- **30分ごと**（Cron-job.org「サイボウズBot」ジョブが起動）

## 何をするか

1. サイボウズにログインして、未承認の日報・週報・設備使用許可書をチェック
2. AI（Claude）がコメントを添削・生成して自動承認

## もうひとつの大事な役割：月次ジョブの「時計係」

Cron-job.org の無料枠（12件）を節約するため、30分ごとに動くこのボットが
「今日は何日・何時か」を確認して、以下の月次ジョブを起動しています（毎時00分の回のみ）：

| 起動するジョブ | タイミング | リポジトリ |
|---|---|---|
| レセコンCSV突合・歩合計算 | 毎月1日 9:00 | apotool-automation |
| 勤怠スプシ自動集計 | 毎月5日 21:00 | kintai-automation |
| 給与振込チェック | 毎月10〜14日 9:00 / 13:00 / 18:00 | payroll-check-bot |

## エラーが起きたら

- GitHubの「Actions」タブで❌赤印の実行を確認
- Claude Codeに「サイボウズBotのログを見て」と頼めば調査します
